Welcome to the Reach Tutorials! You can complete one or more of these tutorials through self study. Reach also offers scheduled or ad hoc webinars, and instructor-led training. See the Schedule or Contact us.

Run a DApp

Run a Reach Hello World decentralized application (DApp) in a POSIX-compliant shell.

Level: Beginner. Duration: 15 minutes.
Set up an IDE

Set up a Reach development environment for Mac, Linux, and Windows.

Level: Beginner. Duration: 15 minutes.
Wisdom for Sale

Build a command-line version and a webapp version of an application that enables two participants, a seller and a buyer, to trade wisdom for tokens via a smart contract running on a private Algorand, Ethereum, or Conflux consensus network residing in a Docker container on your computer.

Level: Beginner. Duration: 2 hours.
Market Day

Build an application that demonstrates the use of strings, integers, booleans, objects, and arrays in interact objects by modeling a conversation and purchase between a seller and a buyer

Level: Beginner. Duration: 1.5 hours.
Coin Flipper

Build an application that enables two participants to bet on the flip of a coin.

Level: Beginner. Duration: 1.5 hours.
Rock Paper Scissors

Build an application that demonstrates several key aspects of the Reach programming language by enabling a game of Rock Paper Scissors between two participants.

Level: Beginner. Duration: 4 hours.
Crypto Fund Me

Build an application that enables a fundraiser to deploy a smart contract that represents a fundraising project and allows various sponsors to contribute to the project repeatedly.

Level: Beginner. Duration: 2 hours.
