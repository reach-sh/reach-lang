4.18 Workshop: Simple NFT

This workshop assumes that you have recently completed race.

In this workshop, we implement a trivial non-fungible token (NFT) where a single creator creates a unique item and initial owns it. The creator and all subsequent owners may transfer ownership of the unique item to a different owner and so on. This workshop uses a participant class to represent owners and is a kind of trial temple before we explore more interesting and in-depth variants of the NFT concept.

This page is a placeholder for a future more detailed workshop. You could try to implement yourself though, given the sketch above! If you’d like to see a draft version of our code, please visit examples/nft-dumb in our GitHub repository
